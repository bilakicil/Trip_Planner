## Functional Requirement

| Kode  | Kebutuhan Fungsional                         | Penjelasan                                                                                          |
| ----- | -------------------------------------------- | --------------------------------------------------------------------------------------------------- |
| FR-01 | Login menggunakan akun Google                | Sistem memungkinkan pengguna melakukan login menggunakan akun Google melalui mekanisme OAuth2.    |
| FR-02 | Menginput preferensi perjalanan              | Sistem memungkinkan pengguna untuk memasukkan preferensi perjalanan seperti kota atau negara tujuan. |
| FR-03 | Mengirim preferensi ke AI                    | Sistem mengirimkan data preferensi perjalanan ke model AI untuk menghasilkan rekomendasi trip.     |
| FR-04 | Menampilkan rekomendasi perjalanan           | Sistem menampilkan rekomendasi perjalanan yang mencakup destinasi wisata dan kebutuhan administrasi. |
| FR-05 | Menampilkan estimasi biaya perjalanan        | Sistem menampilkan estimasi biaya perjalanan yang meliputi transportasi, akomodasi, dan kebutuhan lainnya. |
| FR-06 | Menampilkan rekomendasi akomodasi            | Sistem memberikan rekomendasi penginapan atau akomodasi sesuai dengan destinasi yang dipilih.      |
| FR-07 | Menyimpan rekomendasi perjalanan             | Sistem menyimpan hasil rekomendasi perjalanan dari AI ke dalam database agar dapat dilihat kembali. |
| FR-08 | Penanganan kesalahan dari AI                 | Sistem menampilkan pesan kesalahan yang informatif dan ramah pengguna jika layanan AI gagal merespons. |
